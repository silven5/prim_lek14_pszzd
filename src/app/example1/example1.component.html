Вбудовані pipe

Без форматування:

{{ myDate }}

З форматуванням та парметрами:

{{ myDate | date : "dd/MM/yyyy" }}

Без форматування:

{{ myNumber }}

Форматування два знаки до коми, два після:

{{ myNumber | number : "1.1-2" }}

Інше форматування:

{{ myNumber | number : "3.5-5" }}

Без форматування:

{{ myMoney }}

З форматуванням та парметрами:

З кодом та символом:

{{ myMoney | currency : "UAH" : "symbol-narrow" }}

Інше форматування:

{{ myMoney | currency : "UAH" : "symbol-narrow" : "1.1-1" }}

Без форматування:

{{ myText }}

Ланцюжок:

{{ myText | slice : 6 : 11 | uppercase }}
